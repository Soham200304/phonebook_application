A **Flutter-based Phonebook mobile application** developed as a **Data Structures and Algorithms (DSA) project**.

## ðŸŽ¯ Project Overview

Managing contacts efficiently requires proper data organization and fast operations.  
This Phonebook Application allows users to:

- Add new contacts
- View all saved contacts
- Search contacts by name or phone number
- Sort contacts alphabetically (ascending & descending)
- Update contact details
- Delete contacts using swipe gestures
- Store data persistently using Firebase Firestore

The project emphasizes **DSA logic implementation** behind these operations while providing a user-friendly mobile interface.
